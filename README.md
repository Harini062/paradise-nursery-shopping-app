# Paradise Nursery Shopping App

This is a shopping cart application for an online plant shop. The application showcases a variety of house plants available for purchase. With this application, users can select plants in an online plant shop and add them to the shopping cart.

## Features

- Home page with a welcome message and information about the nursery.
- Product listing page with various plant categories.
- Add to Cart functionality.
- Shopping cart page displaying items in the cart, allowing quantity adjustments, and calculating total costs.
